# How to debug Typescript while using Neovim

```bash
tsc && node inspect main.js example.txt
```

Then make sure that somewhere in your code you have the `debugger` keyword used.

![screen1](./screenshot.png)
It will look like this, make sure to expose source-maps in the `tsconfig.json`.

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "sourceMap": true
  }
}
```
