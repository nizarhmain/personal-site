# How to debug from rails from the console

Imagine you have this

```ruby
class Dog
  def speak
    "woof"
  end
end
```

```ruby
module LoudDog
  def speak
    "WOOF!!!"
  end
end

Dog.prepend(LoudDog)

Dog.new.speak
# => "WOOF!!!"
```
